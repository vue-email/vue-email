## Usage
Add the component to your email template. Include styles where needed.

```vue
<template>
  <e-container>
    <e-button href="https://vuejs.org" style="color: #61dafb">Click</e-button>
  </e-container>
</template>

<script setup>
import { EButton, EContainer } from 'vue-email';
</script>
```
