## Usage
Add the component to your email template. Include styles where needed.

```vue
<template>
  <e-section>
    <e-column>A</e-column>
    <e-column>B</e-column>
    <e-column>C</e-column>
  </e-section>
</template>

<script setup>
import { ESection, EColumn } from 'vue-email';
</script>
```
