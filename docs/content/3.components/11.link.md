## Usage
Add the component to your email template. Include styles where needed.

```vue
<template>
  <e-link href="https://vuejs.org">Go to vue</e-link>
</template>

<script setup>
import { ELink } from 'vue-email';
</script>
```

## Props

::component-props
---
