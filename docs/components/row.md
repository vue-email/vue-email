# Row
Display a block that separates content areas horizontally in your email.

## Getting started
Add the component to your email template. Include styles where needed.

```vue
<template>
  <e-row>
    <e-text>My simple text</e-text>
  </e-row>
</template>

<script setup>
import { ERow, EText } from 'vue-email';
</script>
```
